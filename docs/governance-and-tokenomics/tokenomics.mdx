## ZTR Token

ZTR is ZentraFi's native protocol token on Pharos.

| Property | Value |
|----------|-------|
| Name | Zentra |
| Symbol | ZTR |
| Chain | Pharos |
| Standard | ERC-20 |

## Token Utility

### Governance (veZTR)

Stake ZTR to accrue veZTR voting power:

- Direct liquidity emissions to pools
- Vote on protocol parameters
- Influence treasury decisions

See [Governance](/docs/governance-and-tokenomics/governance) for full details on veZTR mechanics.

### Revenue Sharing (sZTR)

Stakers earn sZTR—a share of real protocol revenue from:

- Trading fees (Zentra X)
- Launch fees (Zentra App)

See [Farming & Rewards](/docs/governance-and-tokenomics/farming) for yield strategies.

### Incentive Alignment

- LPs earn ZTR emissions for providing liquidity
- Projects can create voting rewards to attract liquidity
- Long-term stakers have more influence via veZTR
