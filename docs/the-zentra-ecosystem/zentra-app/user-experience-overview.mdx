## What is Zentra App?

Zentra App is ZentraFi's token launchpad — a no-code platform for launching tokens on Pharos.

It provides a streamlined interface for both project creators and participants.

## For Project Creators

### Launch Configuration

- Set token parameters (supply, name, symbol)
- Choose launch type (fair launch, bonding curve)
- Configure pricing and distribution rules
- Define liquidity allocation

### Launch Management

- Monitor participation in real-time
- Track price discovery progress
- View post-launch analytics

## For Participants

### Discovering Launches

- Browse upcoming and active launches
- View project details and tokenomics
- Check participation requirements

### Participating

- Connect wallet
- Choose contribution amount
- Confirm transaction
- Receive tokens after launch completes

### Post-Launch

- Trade tokens on Zentra X
- Provide liquidity to earn fees
- Track portfolio performance

## Design Principles

- **Simplicity** — No technical knowledge required
- **Transparency** — All parameters visible before participation
- **Fairness** — Adaptive pricing reduces manipulation
- **Integration** — Direct connection to Zentra X trading
