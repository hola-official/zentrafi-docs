## Zentra X

Zentra X is ZentraFi's decentralized exchange — a DLMM-powered trading platform built natively on Pharos.

## Core Purpose

Zentra X provides:

- Efficient token swaps with minimal slippage
- Concentrated liquidity pools for capital efficiency
- Dynamic fees that respond to market conditions
- Deep integration with Zentra App launches

## How It Differs from Traditional DEXs

| Traditional AMM | Zentra X (DLMM) |
|-----------------|-----------------|
| Liquidity spread across all prices | Liquidity concentrated in active bins |
| Constant fees | Dynamic fees based on volatility |
| Slippage on every trade | Zero slippage within bins |
| Lower capital efficiency | Higher capital efficiency |

## Built on Liquidity Book

Zentra X uses Liquidity Book DLMM technology — the same proven system powering platforms like Trader Joe and Merchant Moe.

This gives ZentraFi access to:

- Battle-tested smart contracts
- Optimized trading mechanics
- Proven security model
