## v0.1.0 — Initial release

**Released:** March 2026

### Features

- Three display modes: `Integrated`, `Modal`, `Widget`
- Full theme system with 10 configurable tokens and 25+ derived CSS variables
- Independent wallet mode (built-in wagmi v2 + RainbowKit v2) or external wallet passthrough
- Token list with search, favourites (⭐), and custom token import by contract address
- Slippage tolerance settings (presets: 0.1%, 0.5%, 1.0% + custom)
- Auto-refreshing quotes every 10 seconds with a circular countdown timer
- Price impact warnings (≥5% impact shows red button + alert banner)
- Wrap / unwrap support for PHRS ↔ WPHRS
- ERC-20 approval flow with `maxUint256` single-approval (no per-swap approvals)
- `onSwapSuccess` and `onError` callbacks
- CDN / IIFE build (`window.ZentraX`) with auto-init via `data-zentra-terminal` attribute
- PWA-ready demo at [terminal.zentrafi.xyz](https://terminal.zentrafi.xyz)
