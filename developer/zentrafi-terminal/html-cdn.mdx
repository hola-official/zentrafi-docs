description: "Embed ZentraFi Terminal into any website with just a script tag — no build step required."
---

The CDN build bundles everything (React, wagmi, viem, RainbowKit) into a single IIFE file. Load it with a `<script>` tag and call `window.ZentraX.init()`.

## CDN URLs

```html
<!-- Styles -->
<link rel="stylesheet" href="https://cdn.zentrafi.xyz/terminal/style.css" />

<!-- Bundle (exposes window.ZentraX) -->
<script src="https://cdn.zentrafi.xyz/terminal/main.js"></script>
```

<Warning>
The CDN bundle is ~6.8 MB unminified / ~1.6 MB gzip. It includes React, wagmi, viem, and RainbowKit. If your site already uses these libraries, use the NPM package instead to avoid loading duplicates.
</Warning>

## window.ZentraX API

### `init(options)`

Creates a new terminal instance inside a DOM container.

```ts
window.ZentraX.init({
  containerId: string,           // required — ID of the target <div>
  displayMode?: "Integrated" | "Modal" | "Widget",
  themeType?: "Dark" | "Light",
  theme?: { ... },               // same as initProps.theme
  independentWallet?: boolean,
  walletConnectProjectId?: string,
  appName?: string,
  defaultPair?: { from: string, to: string },
  initialSlippage?: number,
  widgetPosition?: "bottom-right" | "bottom-left" | "top-right" | "top-left",
  widgetSize?: "default" | "small",
  showBranding?: boolean,
  logoUrl?: string,
  onSwapSuccess?: (txHash: string) => void,
  onError?: (error: Error) => void,
})
```

### `update(containerId, options)`

Re-renders an existing instance with merged options. The wallet connection is preserved — no remount occurs.

```js
window.ZentraX.update("swap-terminal", {
  themeType: "Light",
  theme: { primary: "#6366f1" },
})
```

### `destroy(containerId?)`

Unmounts and cleans up a terminal instance.

```js
// Destroy a specific instance
window.ZentraX.destroy("swap-terminal")

// Destroy all instances
window.ZentraX.destroy()
```

## Manual init example

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>My dApp</title>
  <link rel="stylesheet" href="https://cdn.zentrafi.xyz/terminal/style.css" />
</head>
<body>

  <!-- Swap container -->
  <div id="swap-terminal"></div>

  <script src="https://cdn.zentrafi.xyz/terminal/main.js"></script>
  <script>
    window.ZentraX.init({
      containerId: "swap-terminal",
      displayMode: "Integrated",
      themeType: "Dark",
      walletConnectProjectId: "YOUR_PROJECT_ID",
      defaultPair: { from: "PHRS", to: "USDC" },
      onSwapSuccess: function(txHash) {
        console.log("Swap confirmed:", txHash)
      },
    })
  </script>

</body>
</html>
```

## Auto-init (zero JavaScript)

Add `data-zentra-terminal` to any element and the terminal initialises automatically when the page loads — no JavaScript needed.

```html
<div
  data-zentra-terminal
  data-mode="Widget"
  data-theme="Dark"
  data-position="bottom-right"
  data-wallet-connect-id="YOUR_PROJECT_ID">
</div>

<script src="https://cdn.zentrafi.xyz/terminal/main.js"></script>
```

Supported `data-*` attributes:

| Attribute | Corresponding prop | Example values |
|-----------|-------------------|---------------|
| `data-mode` | `displayMode` | `Integrated`, `Modal`, `Widget` |
| `data-theme` | `themeType` | `Dark`, `Light` |
| `data-position` | `widgetPosition` | `bottom-right`, `top-left` |
| `data-wallet-connect-id` | `walletConnectProjectId` | `abc123...` |

<Info>
If no `id` attribute is present on the element, the terminal auto-generates a unique ID. You can still call `window.ZentraX.destroy()` (no argument) to unmount all instances.
</Info>

## Versioned CDN URLs

Pin to a specific version to prevent unexpected breaking changes:

```html
<!-- Pinned to v0.1.0 -->
<link rel="stylesheet" href="https://cdn.zentrafi.xyz/terminal/v0.1.0/style.css" />
<script src="https://cdn.zentrafi.xyz/terminal/v0.1.0/main.js"></script>

<!-- Always latest -->
<link rel="stylesheet" href="https://cdn.zentrafi.xyz/terminal/latest/style.css" />
<script src="https://cdn.zentrafi.xyz/terminal/latest/main.js"></script>
```

## Dynamically switching themes

```html
<button onclick="setLight()">Light mode</button>
<button onclick="setDark()">Dark mode</button>

<script>
  function setLight() {
    window.ZentraX.update("swap-terminal", { themeType: "Light" })
  }
  function setDark() {
    window.ZentraX.update("swap-terminal", { themeType: "Dark" })
  }
</script>
```
