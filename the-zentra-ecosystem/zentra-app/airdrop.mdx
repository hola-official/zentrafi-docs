Zentra App includes an Airdrop tool for distributing tokens or native currency to many recipients at once — perfect for community rewards, team distributions, or marketing campaigns.

## What You Can Airdrop

| Type | Description |
|------|-------------|
| **Native Currency** | Distribute native tokens to multiple wallets |
| **ERC-20 Tokens** | Distribute any token to multiple wallets |

---

## How It Works

### 1. Prepare Your List

Create a list of recipient addresses and amounts:

```
0x123...abc → 1,000 tokens
0x456...def → 500 tokens
0x789...ghi → 2,500 tokens
```

### 2. Approve & Send

- For token airdrops: Approve the total amount
- Pay the service fee
- Confirm the transaction

### 3. Done

All recipients receive their tokens in a single transaction. Any excess funds are automatically refunded to you.

---

## Features

### Batch Distribution

Send to up to **1,000 recipients** in one transaction. No need to send individually and pay gas each time.

### Flexible Amounts

Each recipient can receive a different amount — useful for tiered rewards or proportional distributions.

### Automatic Refunds

If you send more ETH than needed, the excess is automatically returned to your wallet.

## Tips

<Info>
**Before airdropping:**
- Double-check all recipient addresses
- Verify the amounts are correct
- Ensure you have enough tokens + gas + service fee
</Info>

<Warning>
Airdrops are irreversible. Once sent, tokens cannot be recovered if sent to wrong addresses.
</Warning>

## Creating an Airdrop

1. Go to Airdrop on Zentra App
2. Select airdrop type (Native Currency or Token)
3. Enter or upload recipient list (address + amount)
4. Review totals and fees
5. Approve token spending (for token airdrops)
6. Confirm transaction
